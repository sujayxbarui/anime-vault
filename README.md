# Anime Vault — Personal Anime Tracker

> Search, discover, and track your anime — all in one place. No account needed.

---

## Project Purpose

Anime Vault is a personal anime tracking web app. Most anime fans watch dozens of shows but have no easy way to track what they're watching, completed, or planning to watch without creating accounts on external sites.

This app solves that — search any anime, see full details, and manage your personal watchlist right in the browser. Everything saves locally, no login required.

---

## API Used

Jikan API v4 — Unofficial MyAnimeList API  
Link: https://api.jikan.moe/v4  
Docs: https://docs.api.jikan.moe/  
API Key Required: None — completely free and open

---

## Features Planned

- Search — Search anime by title in real-time
- Filter — Filter by genre, type (TV/Movie/OVA), and airing status
- Sort — Sort results by score, popularity, title, or episode count
- My List — Add anime to Watching / Completed / Plan to Watch / Dropped
- Detail View — Full anime info: synopsis, studio, cast, score, air dates
- Stats — Total anime tracked, episodes watched, average score
- Dark / Light Mode — Theme toggle saved in browser
- Export — Download your list as a JSON file

---

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Fetch API
- Array HOFs — filter, map, sort, reduce, find, forEach
- localStorage for watchlist and theme preference
